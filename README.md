# Rock-Paper-Scissors Game (C++)

A simple Rock Paper Scissors game built using C++.  
The player competes against the computer for multiple rounds.

## Features
- Play multiple rounds
- Random computer choice
- Score tracking (User / Computer / Draw)
- Final game result
- Colored terminal output
